🌐 Vishwa T - IT Student Portfolio Website
📌 Project Overview

This is a modern and responsive personal portfolio website developed using HTML, CSS, and JavaScript. The portfolio showcases my skills, education, projects, achievements, and contact information in a professional and visually appealing way.

The website is designed with a modern dark theme, glassmorphism effects, smooth animations, and responsive layouts to provide an excellent user experience across all devices.
